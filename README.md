# Core Finance

## Projeto

O Core Finance é uma plataforma web desenvolvida para auxiliar empresas credoras e de cobrança na recuperação de crédito através da renegociação digital de dívidas.

O sistema permite que clientes consultem suas dívidas, simulem acordos, realizem negociações e acompanhem pagamentos, enquanto administradores gerenciam clientes, empresas credoras, dívidas e acordos.

## Funcionalidades

### Cliente
- Login seguro
- Consulta de dívidas
- Simulação de parcelamento
- Visualização de propostas
- Geração de Pix e boleto
- Histórico de acordos
- Acompanhamento de pagamentos

### Administrador
- Dashboard gerencial
- Gestão de clientes
- Gestão de empresas credoras
- Gestão de dívidas
- Gestão de propostas
- Gestão de acordos
- Gestão de pagamentos
- Relatórios financeiros

- Área do Cliente

## Objetivo

Modernizar o processo de recuperação de crédito através de uma plataforma digital intuitiva, segura e eficiente.
